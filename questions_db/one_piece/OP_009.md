**Question :** Quel est le surnom de Donquixote Doflamingo ?

- A: Le Chirurgien de la Mort
- B: Le Flamant Celeste
- C: Joker
- D: Le Corsaire Fou

**Bonne reponse :** C

**Explication :** Doflamingo est connu sous le nom de code "Joker" dans le monde souterrain, ou il dirige un immense reseau de trafic d'armes et de fruits du demon.
